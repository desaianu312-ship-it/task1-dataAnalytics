Titanic Data Cleaning & Structural Validation
# Overview

This project was completed as Task 1 during my Data Analytics Internship.

The objective was to clean and validate the Titanic dataset by handling missing values, checking duplicate records, and ensuring data consistency for reliable analysis.

#Dataset:

Dataset: Titanic Dataset (Kaggle)/
Format: CSV/
Tool Used: Microsoft Excel/

# Data Cleaning Performed:

Missing Values
Filled 177 missing Age values using the average age.
Filled 2 missing Embarked values with the most frequent value (S).
Replaced 687 missing Cabin values with "Unknown".
Duplicate Records
Checked for duplicate records.
No duplicate rows were found.
Structural Validation
Verified PassengerId uniqueness.
Checked for missing values after cleaning.
Validated Age and Fare columns for invalid values.
Ensured consistent categorical data.
